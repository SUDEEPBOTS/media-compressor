# Contributing

Thank you for your interest in contributing to Media Compressor Pro!

### Bug Reports
- Open an issue describing the bug.
- Include OS and Browser details.
- Provide a sample file if the compression failed.

### Pull Requests
1. Fork the repo.
2. Create your feature branch.
3. Commit your changes.
4. Push to the branch.
5. Open a PR.

By contributing, you agree that your code will be licensed under the MIT License.
